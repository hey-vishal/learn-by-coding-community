# Functional requirements for BMS
Our ticket booking service should meet the following requirements:

1. User should be able to list the cities where affiliate cinemas are located.
2. Each cinema can have multiple halls and each hall can run one movie show at a time.
3. Each Movie will have multiple shows.
4. Customers should be able to search movies by their title, language, genre, release date, and city name.
5. Customers should be registered users to be able to book their movie tickets.
6. Once the customer selects a movie, the service should display the cinemas running that movie and its available shows.
7. The customer should be able to select a show at a particular cinema and book their tickets.
8. The service should show the customer the seating arrangement of the cinema hall. The customer should be able to select multiple seats according to their preference.
9. The customer should be able to distinguish between available seats and booked ones.
10. The system should send notifications whenever there is a new movie, as well as when a booking is made or canceled.
11. Customers of our system should be able to pay with credit cards or cash.
12. The system should ensure that no two customers can reserve the same seat.
13. Customers should be able to add a discount coupon to their payment.
14. Customers should be able to view their previous booking history.
